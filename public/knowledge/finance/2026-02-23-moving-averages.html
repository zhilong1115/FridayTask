I. Why Start with Moving Averages?

The Moving Average (MA) is the oldest and most widely used technical indicator. Its core idea is simple: smooth out price noise to reveal the underlying trend.

Almost all trend-following strategies build on some form of MA. Understanding MAs is the gateway to the entire technical indicator ecosystem.

II. Three Core Moving Averages

1. Simple Moving Average (SMA)

Arithmetic mean of the past N periods:

SMA(t, N) = (1/N) Ã— Î£áµ¢â‚Œâ‚€á´ºâ»Â¹ P(t-i)

Characteristic: Equal weighting (1/N). A price from 50 days ago affects the SMA exactly as much as today's price.

Issue: High lag. And when an old outlier "drops off" the window, the SMA jumpsâ€”the "drop-off effect".

2. Exponential Moving Average (EMA)

Exponentially decaying weights give recent data more importance:

EMA(t) = Î± Ã— P(t) + (1-Î±) Ã— EMA(t-1)
Î± = 2/(N+1)

Key insight: EMA has infinite memory with geometrically decaying weights (1-Î±)áµ. For N=20 EMA, Î±â‰ˆ0.095, so today's weight is ~9.5% while 20 days ago is only ~1.3%.

3. Weighted Moving Average (WMA)

WMA(t, N) = Î£áµ¢â‚Œâ‚€á´ºâ»Â¹ (N-i)Ã—P(t-i) / [N(N+1)/2]

Linearly decreasing weights. Less common in practice but conceptually bridges SMA and EMA.

III. Golden Cross & Death Cross Strategy

  • Golden Cross: Short MA (50d) crosses above Long MA (200d) â†’ Buy
  • Death Cross: Short MA crosses below Long MA â†’ Sell

On the S&P 500 (1960-2020), this strategy returned ~118%, but Buy & Hold outperformed in total return. The real value is drawdown reductionâ€”it helped avoid the worst of the 2008 crash.

âš ï¸ Practical Pitfalls:
  • Whipsaw: In ranging markets, frequent false crossovers eat into profits via transaction costs
  • Severe lag: The 200-day MA turns long after the trend is established
  • Parameter overfitting: 50/200 is convention, not theoretically optimal

IV. Beyond Simple Crossovers

  • DEMA/TEMA: Double/Triple EMA for reduced lag
  • Slope filter: Only accept golden cross when 200MA slope is positive
  • Price-MA distance: Mean reversion when price deviates far from MA
  • Multi-timeframe confirmation: Daily cross + Weekly uptrend = stronger signal

import numpy as np
import pandas as pd
import yfinance as yf
import matplotlib.pyplot as plt

# 1. Fetch data / è·å–æ•°æ®
spy = yf.download("SPY", start="2015-01-01", end="2026-02-01")
close = spy["Close"].squeeze()

# 2. Calculate moving averages / è®¡ç®—å„ç±»å‡çº¿
def sma(series, n):
    """Simple Moving Average"""
    return series.rolling(window=n).mean()

def ema(series, n):
    """Exponential Moving Average"""
    return series.ewm(span=n, adjust=False).mean()

def wma(series, n):
    """Weighted Moving Average"""
    weights = np.arange(1, n + 1)
    return series.rolling(window=n).apply(
        lambda x: np.dot(x, weights) / weights.sum(), raw=True
    )

# 3. Golden Cross Strategy / é‡‘å‰ç­–ç•¥
def golden_cross_backtest(prices, short_window=50, long_window=200,
                           ma_type="sma"):
    """
    Backtest Golden Cross / Death Cross strategy.
    Returns a DataFrame with signals and equity curve.
    """
    ma_func = {"sma": sma, "ema": ema, "wma": wma}[ma_type]
    
    df = pd.DataFrame({"price": prices})
    df["ma_short"] = ma_func(prices, short_window)
    df["ma_long"] = ma_func(prices, long_window)
    
    # Signal: 1 when short > long, 0 otherwise
    df["signal"] = (df["ma_short"] > df["ma_long"]).astype(int)
    
    # Detect crossovers / æ£€æµ‹äº¤å‰ç‚¹
    df["crossover"] = df["signal"].diff()
    # +1 = golden cross, -1 = death cross
    
    # Strategy returns (enter next day) / ç­–ç•¥æ”¶ç›Šï¼ˆæ¬¡æ—¥å…¥åœºï¼‰
    df["daily_ret"] = prices.pct_change()
    df["strat_ret"] = df["signal"].shift(1) * df["daily_ret"]
    
    # Equity curves / èµ„é‡‘æ›²çº¿
    df["buy_hold"] = (1 + df["daily_ret"]).cumprod()
    df["strategy"] = (1 + df["strat_ret"]).cumprod()
    
    return df

# 4. Run backtest / è¿è¡Œå›æµ‹
result = golden_cross_backtest(close, 50, 200, "sma")

# 5. Performance metrics / ç»©æ•ˆæŒ‡æ ‡
def calc_metrics(df):
    strat = df["strat_ret"].dropna()
    bh = df["daily_ret"].dropna()
    
    ann = 252
    
    def sharpe(r):
        return r.mean() / r.std() * np.sqrt(ann) if r.std() > 0 else 0
    
    def max_dd(equity):
        peak = equity.cummax()
        dd = (equity - peak) / peak
        return dd.min()
    
    print(f"Strategy Sharpe:  {sharpe(strat):.3f}")
    print(f"Buy&Hold Sharpe:  {sharpe(bh):.3f}")
    print(f"Strategy MaxDD:   {max_dd(df['strategy']):.2%}")
    print(f"Buy&Hold MaxDD:   {max_dd(df['buy_hold']):.2%}")
    print(f"Strategy Return:  {df['strategy'].iloc[-1]-1:.2%}")
    print(f"Buy&Hold Return:  {df['buy_hold'].iloc[-1]-1:.2%}")
    
    # Count trades / äº¤æ˜“æ¬¡æ•°
    trades = (df["crossover"].abs() == 1).sum()
    print(f"Total Crossovers: {trades}")

calc_metrics(result)

# 6. Comparison: SMA vs EMA Golden Cross / å¯¹æ¯” SMA å’Œ EMA é‡‘å‰
for mt in ["sma", "ema"]:
    r = golden_cross_backtest(close, 50, 200, mt)
    final = r["strategy"].iloc[-1] - 1
    print(f"{mt.upper()} 50/200 Return: {final:.2%}")
