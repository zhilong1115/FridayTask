1. What Are Bollinger Bands?

Invented by John Bollinger in the 1980s, Bollinger Bands are one of the most elegant technical indicators. The core idea: prices tend to revert to their mean, and extreme deviations are statistically rare.

Three lines:

  • Middle Band: N-period SMA (typically N=20)
  • Upper Band: Middle + K Ã— StdDev (typically K=2)
  • Lower Band: Middle âˆ’ K Ã— StdDev

Why 2 standard deviations? Under normal distribution, ~95.4% of observations fall within Â±2Ïƒ. A touch of the band signals a statistically "extreme" price.

2. Key Properties

Adaptive Nature: Unlike fixed-width channels, Bollinger Bands expand during volatile periods and contract during calm ones. They "breathe" with the market.

Bandwidth: Measures the relative width.

BandWidth = (Upper âˆ’ Lower) / Middle = 2K Ã— Ïƒ / SMA

%B Indicator â€” Where price sits within the bands:

%B = (Price âˆ’ Lower) / (Upper âˆ’ Lower)
  • %B > 1: Above the upper band
  • %B = 0.5: At the middle band
  • %B < 0: Below the lower band

3. Three Classic Strategies

Strategy 1: Mean Reversion (Bollinger Bounce)

Assumption: Prices snap back like a rubber band.

  • Buy: Price touches/crosses lower band, re-enters from below
  • Sell: Price touches/crosses upper band, re-enters from above
  • Best for: Range-bound, sideways markets

Strategy 2: Breakout (Bollinger Squeeze)

Key insight: Volatility is mean-reverting. Low vol â†’ high vol explosion.

  • When BandWidth hits N-period low ("Squeeze"), a big move is coming
  • Breakout above upper band â†’ long; below lower â†’ short
  • Caveat: Squeeze predicts magnitude, not direction

Strategy 3: Walking the Band (Trend Following)

In strong trends, price "walks" along the upper or lower band. Mean reversion strategies get destroyed here.

âš ï¸ Critical Trap: Beginners short at the upper band and buy at the lower. In trending markets, this leads to repeated losses. Band touches â‰  reversal signals!

4. Mathematical Depth

Rolling standard deviation captures recent price dispersion. But remember from Day 4:

  • Financial returns exhibit volatility clustering (GARCH effects)
  • Fat tails mean the "95% within Â±2Ïƒ" assumption is violated
  • In practice, band touches occur more frequently than the normal distribution predicts â€” which actually generates more trading signals

5. Connections to Previous Topics

Bollinger Bands + RSI (Day 9) is a classic combo:

  • Price at lower band + RSI < 30 = stronger oversold signal
  • Price at upper band + RSI > 70 = stronger overbought signal
  • Bollinger Squeeze + MACD crossover (Day 10) = directional confirmation

def bollinger_bands(df, window=20, num_std=2):
    """Calculate Bollinger Bands, %B, and BandWidth."""
    df = df.copy()
    df['SMA'] = df['Close'].rolling(window).mean()
    df['STD'] = df['Close'].rolling(window).std()
    df['Upper'] = df['SMA'] + num_std * df['STD']
    df['Lower'] = df['SMA'] - num_std * df['STD']
    df['%B'] = (df['Close'] - df['Lower']) / (df['Upper'] - df['Lower'])
    df['BandWidth'] = (df['Upper'] - df['Lower']) / df['SMA']
    return df

# â”€â”€ 2. Bollinger Bounce å‡å€¼å›å½’ç­–ç•¥ â”€â”€â”€â”€â”€â”€â”€â”€
def bollinger_bounce_strategy(df, entry_pctb=0.0, exit_pctb=0.5):
    """
    Mean reversion: buy when %B < entry_pctb, sell when %B > (1 - entry_pctb).
    Exit at middle band (%B â‰ˆ 0.5).
    """
    df = df.copy()
    df['Signal'] = 0
    position = 0  # 0=flat, 1=long, -1=short
    
    for i in range(1, len(df)):
        pctb = df['%B'].iloc[i]
        if position == 0:
            if pctb < entry_pctb:       # è§¦åŠä¸‹è½¨ â†’ åšå¤š
                df.iloc[i, df.columns.get_loc('Signal')] = 1
                position = 1
            elif pctb > (1 - entry_pctb):  # è§¦åŠä¸Šè½¨ â†’ åšç©º
                df.iloc[i, df.columns.get_loc('Signal')] = -1
                position = -1
        elif position == 1:
            if pctb >= exit_pctb:        # å›åˆ°ä¸­è½¨ â†’ å¹³ä»“
                df.iloc[i, df.columns.get_loc('Signal')] = 0
                position = 0
            else:
                df.iloc[i, df.columns.get_loc('Signal')] = 1
        elif position == -1:
            if pctb <= exit_pctb:
                df.iloc[i, df.columns.get_loc('Signal')] = 0
                position = 0
            else:
                df.iloc[i, df.columns.get_loc('Signal')] = -1
    return df

# â”€â”€ 3. Bollinger Squeeze æ£€æµ‹ â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€
def detect_squeeze(df, lookback=120):
    """Detect Squeeze: BandWidth at N-period low."""
    df = df.copy()
    df['BW_Min'] = df['BandWidth'].rolling(lookback).min()
    df['Squeeze'] = df['BandWidth'] == df['BW_Min']
    return df

# â”€â”€ 4. å›æµ‹ â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€â”€
def backtest(df):
    """Simple vectorized backtest."""
    df = df.copy()
    df['Return'] = df['Close'].pct_change()
    df['Strategy_Return'] = df['Signal'].shift(1) * df['Return']
    df['Cum_Market'] = (1 + df['Return']).cumprod()
    df['Cum_Strategy'] = (1 + df['Strategy_Return']).cumprod()
    
    # Performance metrics
    total_return = df['Cum_Strategy'].iloc[-1] - 1
    sharpe = df['Strategy_Return'].mean() / df['Strategy_Return'].std() * np.sqrt(252)
    max_dd = (df['Cum_Strategy'] / df['Cum_Strategy'].cummax() - 1).min()
    
    print(f"Total Return: {total_return:.2%}")
    print(f"Sharpe Ratio: {sharpe:.2f}")
    print(f"Max Drawdown: {max_dd:.2%}")
    return df
